# Concept Summary

Chazo: Fatebound is a 2D top-down action RPG / dungeon crawler where the player controls Chazo, a young fighter trying to rescue his twin sister Lyra from the Veilborn.

The prototype demo focuses on Mossgrave Outskirts, normal Veilborn enemies, the Brakkor mini-boss, the Eclipse Shard reward, and a final teaser.
