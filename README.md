# Chazo: Fatebound Prototype Demo

WIG3005 Game Development Project  
Semester 2, 2025/2026  

This repository contains the prototype demo for **Chazo: Fatebound**, a 2D top-down action RPG / dungeon crawler.

## Demo Scope

The prototype focuses on a one-zone vertical slice:

- Title screen
- Short story intro
- Mossgrave Outskirts
- Normal Veilborn enemies
- Chazo basic combat and Dawn Arts
- Brakkor mini-boss
- Eclipse Shard reward
- Optional Pendant of Malaya collectible
- Final teaser / demo ending screen

## Workflow Rules

1. Pull before working.
2. Use branches for tasks.
3. Commit small changes.
4. Do not edit the same Godot scene at the same time.
5. Push after finishing a stable change.
6. Test in Godot before pushing.
7. Do not upgrade Godot version alone.
8. Never use force push.
